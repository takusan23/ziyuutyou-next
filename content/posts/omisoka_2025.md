## Xperia 1 VII
`7`世代目`Xperia 1`です。`Xperia 1 V`の電池持ちが悪くなりすぎて`60fps`に戻してたもん。  
回収沙汰になったこの機種、わたしが買った機種ももれなく対象かつ、**文鎮**と化しました。

まあ写真残ってるけど取りに行けば良いか～程度に思っていましたが、数日放置してダメ元で充電器に繋いで、電源ボタンを長押ししたら**奇跡的に復活**しました。大急ぎでバックアップしました。  
写真をサルベージ出来たので`令和7年度版`の記事がかけました（他のスマホで取ったのもあるので最悪それでも良かった）

よく故障を疑うときに**完全放電**を試せって言われるけどこれが効いたことなんて一回もなかったからちょっとうれしかった。

いいところ

- 電池持ちが良い
    - 常に`120fps`にしてる
- `4K`がなくなったけど私はアンチグレアフィルムを貼り付けるのでどっちでも良かった
    - あと画面がめっちゃ明るい、より`HDR`って感じ。
- 広角カメラはあんまり歪んでないイメージ
    - `0.7x`なのでそういうものなのかもしれない
- アスペクト比が変わった案外いいかも
    - ホーム画面が横に広くてびっくり
- `UltraHDR`対応
- `Camera2 API`で`10 ビット HDR`動画が撮影できてるはず

きになるところ

- たかい
- カメラは難しい
    - プレビューをしばらく見てると色温度があってくる、何もしないと白飛びしがち

（以下はアップデートで再現しなくなってる、、気がする）

- おサイフケータイが反応しない
- `Always On Display`から復帰するのに`10`秒くらいかかるときがある

ゲームしないのでこんなにもハイエンドである必要はないのですが、`10ビットHDR + 60fps`動画撮影が欲しいからやっぱハイエンドが良いかも。。

![たーんすたいるもだめ](https://oekakityou.negitoro.dev/original/78aa4978-4ad6-4d0b-9509-ba6fcb91baf0.png)

まあそれでも`Cloudflare`を使ってる気がします。~~一人しかいないのに金払いたくない~~ タダで便利機能を使わせていただいているので文句は言いません。  
オリジン（ここでは`Misskey`サーバーのこと）を無料でいい感じに守ってくれますから。

```plaintext
わたし <-> [Cloudflare] <-> [サーバー nginx <-> Misskey Node.js サーバー]
```

雑で申し訳ないですが、`Cloudflare`をあいだに挟んでいます。間にいるので、サーバーが落ちればさっきのエラー画面で一番下の`Host`が`バツマーク`になる。  
また、`Cloudflare`には`https`にする機能があって、`サーバー <-> Cloudflare 間`は`http`でも、`ブラウザ <-> Cloudflare 間`を`https`にしてくれます。`nginx`に`Let's Encrypt`を入れること無く`https`に出来る。

## 画像配信
このブログ、記事の中身である`マークダウン`ファイルは`git`で管理してますが、画像も管理するとリポジトリがデカくなりすぎるので`Imgur`にアップロードしていました。  
`静的サイト書き出し`あるあるかは分からないですが、画像をどこに置くかは考えないとかなと思います。

ただ、いちいち`GIMP`でサイズを小さくするのも面倒だし、`UltraHDR`画像を記事に貼り付けたくなった（対応してないはず）ので、画像配信を考える必要が出てきた。  
というわけで`AWS`のサーバーレスの**練習問題**であるような画像配信サービス的なのを作ってみました。

```plaintext
私 -> [受付 S3] -> [Lambda を使って ImageMagick を起動し小さくする] -> [配信 S3]
```

```plaintext
みなさんのブラウザ -> [CloudFront] -> [配信 S3]
```

`ImageMagick`を`Lambda`上で動かすので小さくする作業が自動化され、`ImageMagick`自体も`UltraHDR`をサポートするために自前でビルドしたものを使っています。  
（大げさに聞こえるかも、説明書通りにコマンドを叩けば`UltraHDR`サポートありの`ImageMagick`が作れます）

`UltraHDR`も貼れるし画像を小さくする手間もないし最高～  
（なお`Imgur`を辞め自前で配信、加工するので少なからず`AWS`の料金が発生しうる）

これが投入されたのが`令和7年度版`のあの記事。
